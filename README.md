# DSQUARED

> Explore the Unknown

Космический портфолио-сайт с интерактивными анимациями, параллаксом и атмосферным дизайном.

### Features

- Космическая туманность и движущиеся звёзды
- Интерактивная планета с кольцом (следит за мышкой)
- Параллакс-эффект на hero-секции
- Летающие частицы
- Полностью адаптивный дизайн

### Tech Stack

- React 19
- Vite
- CSS Animations

### Run locally

```bash
npm install
npm run dev
```
